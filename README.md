# ChessDate 🎯♟️

A real-time chess-based dating platform where strategy meets romance. Connect with opponents, play competitive chess, and find your perfect match through the ultimate game of minds.

## 🌟 Features

- **Random Matchmaking**: Match with the opposite gender for real-time chess games and live chat.
- **Learn While Dating Mode**: Interactive tactical tips and analysis powered by Gemini AI to help you grow while you connect.
- **Intelligent Bot Fallback**: Human-like bot opponents with weighted difficulties (Easy to Extreme) ensure you always have an opponent.
- **Elo Rating System**: Competitive skill-based matchmaking for balanced and fair gameplay.
- **Credit System**: Sustainable play with an automated credit regeneration system (+1 every 6h).
- **AI Icebreakers**: Dynamic, chess-themed pick-up lines generated by Gemini AI for every match.
- **Social Features**: Follow players, save favorites, and manage a friends list with dedicated private messaging.
- **Modern Romantic UI**: A beautiful, responsive interface with a vibrant pink aesthetic, glassmorphism, and smooth animations.
- **Comprehensive Leaderboard**: Track the top grandmasters in the kingdom with real-time ranking updates.
- **Discovery & Search**: Find specific players by username, display name, or location.
- **Admin Dashboard**: Robust tools for managing users, bot toggles, and system settings.

## 🛠️ Tech Stack

### Frontend
- **React 18** with **TypeScript** for a robust and type-safe UI.
- **TailwindCSS** for a modern "Romantic" aesthetic (pink gradients, glassmorphism).
- **Framer Motion** for smooth, high-quality micro-animations.
- **Chessboardjsx** for a responsive and interactive chess board.
- **Socket.io Client** for low-latency, real-time communication.
- **Zustand** for lightweight and efficient state management.
- **React Query** for optimized server-state fetching and caching.

### Backend
- **Node.js** with **Express** for a high-performance REST API.
- **Socket.io** for real-time WebSocket events and game synchronization.
- **PostgreSQL** with **Sequelize ORM** for reliable persistent storage.
- **Redis** for lightning-fast matchmaking queues and game state caching.
- **Gemini AI** for dynamic content generation (Icebreakers, Tips, Bot personalities).
- **JWT & Bcrypt** for secure authentication and password hashing.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v16 or higher)
- **PostgreSQL** (Database)
- **Redis** (Matchmaking & Caching)
- **Gemini AI API Key** (for AI features)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/chessdate.git
   cd chessdate
   ```

2. **Install dependencies**:
   ```bash
   # In the root directory
   npm install
   cd backend && npm install
   cd ../frontend && npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the `backend/` directory with the following variables:
   ```env
   PORT=4000
   DATABASE_URL=postgres://user:password@localhost:5432/chessdate
   REDIS_URL=redis://localhost:6379
   JWT_SECRET=your_jwt_secret
   GEMINI_API_KEY=your_gemini_api_key
   NODE_ENV=development
   ```

4. **Run the application**:
   ```bash
   # From the root directory
   npm run dev
   ```
